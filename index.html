Aprendendo HTML do Zero

1. O que é HTML?

HTML significa HyperText Markup Language. Ele é usado para criar a estrutura de páginas da web, como textos, imagens, links e listas.

2. Estrutura básica de uma página HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título da Página</title>
  </head>
  <body>
    Aqui vai o conteúdo da página!
  </body>
</html>

3. Principais tags

  • <h1> até <h6>: Títulos (h1 é o maior, h6 o menor)
  • <p>: Parágrafos de texto
  • <a>: Links
  • <img>: Imagens
  • <ul> <li>: Listas

4. Exemplos práticos

Título e parágrafo:

<h1>Bem-vindo ao meu site</h1>
<p>Esse é um parágrafo de exemplo.</p>

Link:

<a href="https://www.google.com">Ir para o Google</a>

Imagem:

<img src="https://via.placeholder.com/150" alt="Exemplo de imagem">

5. Conclusão

O HTML é a base de todo site. Com ele você cria a estrutura. Depois, pode usar CSS para estilos e JavaScript para interatividade.

Aprendendo CSS do Zero

1. O que é CSS?

CSS significa Cascading Style Sheets. Ele é usado para dar estilo ao HTML, como cores, tamanhos, fontes e layouts.

2. Como usar CSS

  • Inline: direto dentro da tag HTML.
  • Interno: dentro da tag <style> no HTML.
  • Externo: em um arquivo separado .css.

3. Exemplos

Exemplo Inline:

<p style="color: blue;">Texto azul</p>

Exemplo Interno:

<style>
  p { color: red; }
</style>
<p>Texto vermelho</p>

4. Propriedades mais usadas

  • color: cor do texto
  • background-color: cor de fundo
  • font-size: tamanho da fonte
  • margin: espaço externo
  • padding: espaço interno
  • border: borda do elemento

5. Conclusão

O CSS transforma um site simples em algo visualmente agradável. Ele trabalha junto com o HTML para criar páginas profissionais.

Aprendendo JavaScript do Zero

1. O que é JavaScript?

JavaScript é a linguagem de programação usada para dar funcionalidade e interatividade aos sites. Com ele você pode criar botões que reagem a cliques, animações, formulários inteligentes e muito mais.

2. Como usar JavaScript

  • Interno: dentro da tag <script> no HTML.
  • Externo: em um arquivo separado .js.

3. Exemplo básico

Um alerta ao abrir a página:

<script>
  alert("Bem-vindo ao meu site!");
</script>

4. Manipulando conteúdo

HTML:

<p id="texto">Texto original</p>
<button onclick="mudarTexto()">Clique aqui</button>

JavaScript:

function mudarTexto() {
  document.getElementById("texto").innerText = "O texto mudou!";
}
